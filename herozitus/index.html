HeroZitus

THE DISCORD MULTI-BOT

ABOUT

HeroZitus is an out-of-this-world Discord multi-bot that brings many unique features to all of your Discord servers.

The main goal Herozitus plans to accomplish is to provide users access to commands that are not commonly found on other Discord bots.

Available To 0 Users

Across 0 Discord Servers

COMMANDS

All of the current commands are:

COMMAND
DESCRIPTION
USE
calc
Calculate a mathematical problem.
$calc (Mathmatical Problem)
covid
Track a country or worldwide COVID-19 cases.
$covid ("country" / "all")
days
Displays the amount of days since the COVID-19 crisis started.
$days
help
Displays all commands and their functions.
$help
impostor
Sends the impostor screen from Among Us with the name placed after the command.
$impostor (any string)
members
Displays the amount of members that are in the current server. Optionally, you can pass 'all' as a parameter and display each member in the server.
$members (OPTIONAL: "all")
ping
How many milliseconds it takes for your message to be recieved by Discord.
$ping
servers
Displays the amount of servers that the bot is in.
$servers
someone
Mentions someone random in the server.
$someone
